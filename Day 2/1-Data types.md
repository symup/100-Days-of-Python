## 1. String
"Hello", "123", ...

     print("123"+"345")

output:

## Subscript
By using [ ] we are able to dissect our string and pull out individual characters when we need it.   

    #print H
    print("Hello"[0])
    
    #print o
    print("Hello"[4])
    
output

    H
    o
